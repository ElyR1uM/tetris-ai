# Tetris-Ai

Python-basiertes Terminal Tetris game

## Setup für trainieren im Codespace:
- Benötigte Python-Version: 3.10
- python -m venv env
- pip install --upgrade pip
- pip install --upgrade keras matplotlib tensorflow
- python trainer.py
